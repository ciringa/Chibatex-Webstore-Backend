# Description 📰

API [Nest](https://github.com/nestjs/nest) com features 

## Features ⬆️

- Login e singup de usuários
- Carrinho de compras
- historico de compras
- Loja de produtos 

# Project setup ⚙️

```bash
$ npm install
```

## Compile and run the project 🚀

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Run Documentation

```bash
#show API documentation
$ npm run start:doc
```
